Thomas's JS Portfolio

In sophmore year of high school, I took a class on web design. In this class, I learned many valuable skills for front-end development, including the basics of HTML and CSS, as well as improved my Javascript skills from what I had learned in previous computer science classes. This class improved not only my hard skills however, but invaluable soft skills, including effective collaboration, communication, mindfulness of deadlines, and, most important of all, the ability to learn and grow efficiently as a programmer.

This computer science class culminated in a final project in which we created a webpage to showcase all of my major projects throughout the year, the result of which you are seeing right now. Although not all of these projects are spectacularly designed or styled, I believe they reflect the progress which I have made throughout the year, and the progress which I will hopefully continue to make in years to come. Thank you for taking the time to check out my portfolio!


About Me

The first project of the year, this project aimed to test our knowledge of HTML structure, including headers, paragraphs, lists, and video embeds, as well as our ability to use CSS styles. The content of the page itself descibed our hobbies, interests, and achievements, as well as basic background information about us. At this point, my HTML skills were rather limited, and I was not able to remove the default margin of the body or create flexboxes to space out the anchors, however despite this I believe I still managed to create a relatively aesthetic pleasing static page with the tools I had at the time.

Global Concern

For our second project, each member of our class chose a global issue which they were passionate about and created a webpage about their issue which was then presented to the rest of the class. The project required anchor elements for internal navigation, as well as a separate CSS file. Most importantly, this project was the first one in which we demonstrated our ability to use division elements, arguably the most important element in HTML. In the process of creating this project, for which I chose deforestation, I learned how to create a sidebar and a header with an image background, two design elements which quickly became some of my favorites to use in following projects.

Collector's Guide

For our next project, we were tasked with creating a 'Collector's Guide', consisting of a main page with links to several other webpages for each object in a collection of our choosing. This project focused on our use of relative links, as well as our ability to work in a group with a classmate. For my and my partner's project, we chose to make a webpage focused around staplers, rating several of the ones found around the school. As mentioned in the previous project, the sidebar and header image background components were also used in this project, along with a similar black-on-grey color scheme. Overall, I believe this project turned out well, showcasing effectively our knowledge of relative links.

Internet Concerns

Our next project was unique in that it incorporated not only the use of our web design skills, namely animations and transitions, but also our knowledge regarding the social and ethical isues surrounding the Internet. I chose to research the issue of computer security, and included some resources for staying safe on the Internet. For the technical requirements, I added subtle transitions upon highlighting links. I again made use of the black-on-grey color scheme, combined with an image background header and a sidebar for ease of navigation. This was improved upon by the use of a gradient to slightly darker shade of grey for the background in order to draw the viewers' eyes downward naturally.

Buzzfeed-Style Quiz

Nearing the end of the year, the second-to-last project had the goal of testing what we had learned about forms in HTML, and, most importantly, Javascript, by having us create a quiz similar to those found on buzzfeed, with different results depending on your answers. For this project, I made a quiz to determine what fruit aligned with your personality based on scientific metrics such as vibes, and which seasons the fruits grow in. Although I had worked with Javascript previously, I had not ever integrated it into HTML, so using the two in conjunction was a fun challenge to which I feel I answered with a satisfactory solution.

Modifiable Cookbook

For our last project of the year (excluding the one you are viewing right now), we were tasked with creating a webpage to showcase recipes, and which would allow the user to add new recipes. Javascript being my 'native language' of programming languages so to speak, I greatly enjoyed this project and the challenges it presented. Although my final product was rather lacking in way of CSS styling, I am happy with the final result and the effort that I put into it (including making pancakes for the pictures!)
